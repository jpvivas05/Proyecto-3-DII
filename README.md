# Proyecto-3-DII
Proyecto de electrónica digital 2. Sensor de temperatura desplegado en pantalla TFT. Sistema de comunicación entre núcleo STM, ESP32 y sensor de temperatura LM75 accionados por botones. Despliegue de valor de temperatura por medio de pantalla interactiva TFT.

Materiales necesarios:
- Núcleo STM F446RE
- ESP32
- 2 Botones
- Pantalla TFT paralela
- Neopixel RGB (Solo uno es necesario)
- Sensor de temperatura LM75 con módulo I2C

En la configuración, es necesario:
- Añadir librería de Neopixels en ESP32
- Añadir librerías FATFS para lectura de SD y uso de pantalla TFT
- Añadir documentos para despliegue de imágenes
